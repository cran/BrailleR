Vignettes from package 'BrailleR'

At present only one file exists for supporting the BrailleR package. This document is an explanation of the project's history and ethos. Hopefully it will inspire collaborations from any like-minded individuals.

The document can be read in either a pdf version (which is bundled with the package), or an html version accessed through the BrailleR Project homepage.
